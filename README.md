# Optimized-Equipment-Utilization-and-Management-for-IVF-Laboratory-Operations

## Overview üìå

This project aims to improve the usage and management of equipment in IVF (In Vitro Fertilisation) laboratories. IVF labs use costly and critical equipment such as incubators, microscopes and ICIS (Intracytoplasmic Sperm Injection) workstations for important medical procedures. Due to poor planning and uneven usage, some equipment remains idle while others are overused, causing delays and increased operational costs.

The project proposes a centralized, data driven system to monitor equipment used and availability. This system helps in better scheduling, reducing downtime and improving overall workflow efficiency.

The objective of the project is to maximize equipment utilization, reduce delays and lower unnecessary operational costs, there by improving the performance of IVF laboratory operations.  


## Dataset üìÅ

- Format : CSV
- Description : The dataset contains structured healthcare-related data used to analyze equipment usage, operational efficiency and key performance indicators within an IVF laboratory environment. It helps identify utilization patterns, downtime and areas for process improvement.

#### 4. SQL Analysis
- Loaded cleaned healthcare data into MySQL. 
- Wrote SQL queries to analyze equipment usage, availability, and performance.

#### 5. Dashboard Development
- Developed an interactive Power BI dashboard to monitor equipment utilization, downtime, and efficiency.

#### 6. Reporting & Presentation
- Created a structured report and presentation highlighting insights and recommendations to improve IVF lab operations.
